Unité de Valeur par Coeur Processeur (PVU)

Le terme "Unité de Valeur par Coeur de Processeur (PVU)" désigne une unité de mesure selon laquelle le Logiciel est concédé sous licence. Le nombre d'Autorisations d'Utilisation pour les Unités de Valeur par Coeur de Processeur requises dépend de la technologie de processeur (déterminée selon le site Web Processor Value Unit (PVU) licensing for Distributed Software à l'adresse https://www.ibm.com/software/passportadvantage/pvu_licensing_for_customers.html) ainsi que du nombre de Coeurs de Processeur auxquels le Logiciel peut accéder. Selon IBM, et dans le cadre de la concession de licence selon le nombre de PVU, un processeur désigne chaque Coeur de Processeur résidant sur une puce. Par exemple, une puce processeur à deux Coeurs comporte deux Coeurs de Processeur.
